The latest update to Spout 2.006 (Update 4) includes a utility program, "SpoutSettings" to set up options for Spout texture sharing as well as demonstration sender and receiver applications that can be used for testing with your system.

SpoutCam

"SpoutCam" is a virtual webcam that is also a Spout receiver. A setup program provides options for frame rate and resolution and can be used to register for both 32 bit and 64 bit host programs. SpoutCam is now a separate repository.

Download the latest release from the repository Releases page below.
Refer to the instructions and "readme.txt" in the archive for further details.

Spout SDK

To make sure you have an up-to-date copy, download or clone the Spout 2.006 SDK from the master branch on GitHub. The repository utility programs can still be used, but the latest versions in Update 4 above are ready for the next 2.007 release.

The latest revisions can be downloaded from the beta branch.
Experimental and testing work can be found in the develop branch.
Existing applications can be tested simply by re-building with the revised SDK files.
New features will become effective with no code changes.
Refer to the SpoutSender and SpoutReceiver classes for new functions.
